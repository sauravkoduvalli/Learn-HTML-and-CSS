### What is Hooks in React?

Hooks in react are special functions introduced in `React 16.8` that allow functional components to have state, use lifecycle methods, optimization, and access other React features previously only available in class components. They were introduced to simplify code, improve reusability, and provide a cleaner alternative to class components.
